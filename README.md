# [DOLAR e Câmbio] - Dashboard 

## Requisitos
 - Python 3.9.9
 - Requests
 - DuckDB
 - Dash 3.3

## Instalação
1. Certifique-se de que tem o Python 3.9.9 instalado
2. Clone o projeto
3. Crie e ative um ambiente virtual:
   
   ```
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
   ```

## Como rodar
1. Abra a pasta em seu cmd
2. No caminho raiz, execute o comando abaixo:
   ```
   python main.py
   ```

1. Aproveite =D

## Justificativa Métricas
- Valor médio ANUAL
-- Entrega uma noção macro de câmbio do DOLAR em 2025
- Variação média mensal Bruta
-- Entrega uma noção do desempenho do câmbio do DOLAR durante todo o período de 2025 em valor bruto
- Variação média mensal em %
-- Entrega uma noção do desempenho do câmbio do DOLAR durante todo o período de 2025 em valor percentual
- Variação média diária bruta
-- Entrega uma noção da volatilidade do câmbio do DOLAR durante todo o período de 2025 em valor bruto
- Variação média diária em %
-- Entrega uma noção da volatilidade do câmbio do DOLAR durante todo o período de 2025 em valor percentual
